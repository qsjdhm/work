# 目录介绍
* src文件夹下面是java代码、WebContent/WEB-INF下是后台配置文件
* WebContent文件夹下面是前端代码
* - admin      后台管理系统代码
* - column     前台页面频道代码
* - common     项目前端一些通用文件
* - staticFile 静态化生成的前台页面

#后端使用技术
* 数据库采用mysql5
* 后台路由采用spring mvc
* 静态化采用freemarker + java流生成文件
* 后台服务采用spring + hibernate来实现快速的搭建整个MVC中的M模块

#后台管理系统页面使用技术
* react + react-redux + antd
* react-router路由按需加载
* webpack-dev-server

#前台页面使用技术
* jquery + bootstrap
* ueditor
